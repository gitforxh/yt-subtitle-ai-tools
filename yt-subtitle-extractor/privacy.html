Privacy Policy for Subtitle Learner

Effective date: 2026-02-19

Subtitle Learner ("the Extension") helps users learn YouTube subtitles line by line with dictionary and AI explanations.

1) What data we process

The Extension may process:

  • Selected subtitle/transcript text from the YouTube video page (for dictionary/AI explanation features).
  • User settings stored locally in the browser, including chosen AI provider, selected model, preferred language, optional helper URL/session key, and user-provided API keys (OpenAI or Google Gemini).
  • Basic request/response data needed to show dictionary and AI results.

2) How data is used

Data is used only to provide Extension features:

  • Show subtitle learning UI on YouTube watch pages.
  • Perform dictionary lookups.
  • Send user-selected text to the chosen AI provider when the user explicitly requests AI explanation.
  • Save preferences so users do not need to reconfigure each session.

3) External services

When users trigger lookup/explanation features, data may be sent to:

  • api.openai.com (OpenAI)
  • generativelanguage.googleapis.com (Google Gemini)
  • api.dictionaryapi.dev (Dictionary API)
  • jisho.org (Japanese dictionary API)
  • 127.0.0.1 / localhost (optional local helper mode)

Each external service is governed by its own privacy terms.

4) Data storage and retention

  • Settings are stored locally using chrome.storage.local.
  • The developer does not operate a central backend for collecting user data.
  • The Extension does not sell personal data.

5) Data sharing

The Extension does not share user data with third parties except as required to fulfill user-requested API calls (dictionary/AI providers listed above).

6) Security

The Extension is designed to minimize permissions and only access required hosts. Users are responsible for protecting their own API keys and browser profile access.

7) Children's privacy

The Extension is not specifically directed to children under 13.

8) Changes to this policy

This policy may be updated from time to time. Updates will be reflected on this page with a new effective date.

9) Contact

For privacy questions, contact the publisher through the Chrome Web Store support/contact channel for this Extension.
